# Компоненты

## ETCD
  * Описание: ```etcd``` — основной исполняемый файл сервера ETCD, реализующий функциональность распределенного надежного хранилища данных в формате ключ-значение.
  * Расположение: ```/usr/local/bin/etcd```

## Etcdctl
  * Описание: ```etcdctl``` — инструмент командной строки для взаимодействия с сервером ETCD, предоставляющий основные функции управления и доступа к данным.
  * Расположение: ```/usr/local/bin/etcdctl```

## Etcdbrctl
  * Описание: ```etcdbrctl``` — утилита для управления резервным копированием, дефрагментацией, компрессией и восстановлением данных ETCD.
  * Расположение: ```/usr/local/bin/etcdbrctl```
