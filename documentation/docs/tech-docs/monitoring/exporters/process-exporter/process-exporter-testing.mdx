kubectl get pods -l app=prometheus-process-exporter --namespace in-cloud-monitoring-nodes

## Доступ к метрикам

Чтобы получить доступ к метрикам, вам нужно получить IP-адрес или имя вашего пода `Process Exporter` и порт, на котором он работает (по умолчанию 9101). Вы можете сделать это с помощью:

```bash
kubectl get endpoints/prometheus-process-exporter --namespace in-cloud-monitoring-nodes  -o json

```
Выберите нужный под, к примеру `prometheus-process-exporter-bwwdg` и запустите:

```bash
kubectl port-forward pod/prometheus-process-exporter-bwwdg 9101:9101 --namespace in-cloud-monitoring-nodes
```

Теперь вы можете перейти по адресу [http://localhost:9101/metrics](http://localhost:9101/metrics) в вашем веб-браузере, чтобы видеть метрики, собираемые `Process Exporter`.
