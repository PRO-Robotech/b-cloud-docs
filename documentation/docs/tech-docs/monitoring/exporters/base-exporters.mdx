# Экспортеры

`Prometheus exporters` — это отдельные программы или компоненты, которые собирают метрики из различных источников и представляют их в формате, понятном для системы мониторинга Prometheus/VictoriaMetrics. Экспортеры позволяют собирать и сохранять данные о состоянии систем, приложений и инфраструктуры.

Существует множество различных экспортеров, каждый из которых предназначен для определённого типа данных. Например:

* `Node Exporter` — собирает метрики с серверов Linux, такие как использование CPU, памяти, нагрузки на диск и сети.
* `Blackbox Exporter` — позволяет мониторить доступность и производительность внешних сервисов с помощью различных методов (HTTP, TCP, ICMP и др.).
* `JMX Exporter` — используется для экспорта метрик из Java-приложений, работающих на платформе Java Management Extensions (JMX).
* Database Exporters — такие как `PostgreSQL Exporter` или `MySQL Exporter`, собирают метрики из баз данных.

Экспортеры значительно упрощают процесс интеграции различных систем и сервисов с `Prometheus` или `VictoriaMetrics`, обеспечивая гибкий и расширяемый подход к мониторингу
