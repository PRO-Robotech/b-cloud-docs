## Доступ к метрикам

Чтобы получить доступ к метрикам, вам нужно получить IP-адрес или имя вашего пода `Node Exporter` и порт, на котором он работает (по умолчанию 9100). Вы можете сделать это с помощью:

```bash
kubectl get endpoints/prometheus-node-exporter --namespace bcloud-monitoring-nodes  -o json

```
Выберите нужный под, к примеру `prometheus-node-exporter-bwwdg` и запустите:

```bash
kubectl port-forward pod/prometheus-node-exporter-bwwdg 9100:9100 --namespace bcloud-monitoring-nodes
```

Теперь вы можете перейти по адресу [http://localhost:9100/metrics](http://localhost:9100/metrics) в вашем веб-браузере, чтобы видеть метрики, собираемые `Node Exporter`.
