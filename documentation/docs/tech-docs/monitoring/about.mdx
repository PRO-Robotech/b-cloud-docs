Данная документация предназначена для инженеров, занимающихся эксплуатацией, развертыванием и настройкой мониторинга на базе `VictoriaMetrics` в платформе B-Cloud, построенной на базе Kubernetes. Основной целью документации является предоставление детализированных инструкций по созданию, настройке и управлению мониторингом в платформе B-Cloud, а также описание ключевых компонентов и механизмов работы экспортеров (prometheus-exporter).

Система мониторинга в платформе B-Cloud является решением с открытым исходным кодом, предназначенным для для остлеживания состояния самой платформы и контейнеризованных приложений в облачной инфраструктуре.

Этот документ не ограничивается только описанием инструментов мониторинга, таких как `VictoriaMetrics` и экспортеры метрик, а предоставляет пошаговое руководство по созданию и настройке компонентов мониторинга платформы B-Cloud с нуля, что позволяет понять внутренние процессы и архитектуру системы.
