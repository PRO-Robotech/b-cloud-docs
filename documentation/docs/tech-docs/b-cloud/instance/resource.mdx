# Ресурс Instance

`Instance` — представляет собой введенную абстракцию, созданную для обеспечения изолированного вычислительного окружения, предназначенного для развертывания и выполнения приложений или сервисов. Основная цель `instance` — предоставить клиенту управляемый экземпляр ресурсов, где можно настраивать конфигурацию рабочих нагрузок, контролировать использование ресурсов и квоты, отслеживать состояние запущенных процессов и интегрироваться с инструментами DevOps, для эффективного управления и масштабирования приложений.
